Program Hakkında:

Program girilen sayının kaç adımda 0'a ulaşacağını adım adım göstererek aşağıdaki kurallara göre hesaplamaktadır:

a)Sayı çift ise 2'ye bölünür ve yeni değer için program devam eder.

b)Sayı tek ise sayıdan 1 çıkarılır ve elde edilen yeni değer üzerinden işlemler devam eder.


Nasıl Çalıştırılır?

1)Belirtilen kurallara göre kaç adımda 0'a ulaşacağının hesaplanması için pozitif bir tamsayı giriniz.

2)Girdiğiniz sayı için yapılan işlemler ve kaç adımda 0'a ulaştığı ekranda görüntülenecektir.
